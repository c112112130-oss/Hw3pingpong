# Hw3pingpong
使用 **有限狀態機（FSM）** 實作一個乒乓球對打遊戲，  
透過 **LED** 顯示球的位置，並使用左右按鍵控制擊球，同時記錄雙方得分。

---
## 功能說明

### LED 顯示（乒乓球模擬）
- LED 燈號代表乒乓球位置
- 球會由左向右或由右向左移動
- 移動方向即為球的飛行方向

### 有限狀態機（FSM）

系統主要包含以下狀態：

- **Moving_Left**  
  球由右往左移動

- **Moving_Right**  
  球由左往右移動

- **Lwin**  
  左方得分，等待左方發球

- **Rwin**  
  右方得分，等待右方發球

FSM 依據 LED 位置與按鍵輸入，判斷是否成功擊球或失誤。

### 按鍵控制
- 玩家需在球移動至最邊緣 LED 時按下按鍵，才算成功擊球
- 若提早按鍵或未及時按下，視為失誤，對方得分

### 得分系統
- 成功得分時，對應玩家分數加一
- 得分後由該方進行發球
- 分數會以 LED 顯示方式呈現

### 速度控制
- 使用除頻器產生較慢的控制時脈
- 當雙方總得分 **≥ 4 分** 時，球速會自動加快
- 增加遊戲挑戰性
